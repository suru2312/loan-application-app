# Loan Application Form

## Features
- 8-step multi-step form
- File upload & signature
- EMI calculation
- Backend integration (Django REST)
- Conditional steps
- Responsive UI

## Tech Stack
- React + Vite
- Django REST Framework
- MySQL

## How to Run

### Backend
cd backend
venv\Scripts\activate
python manage.py runserver

### Frontend
cd loan-app
npm install
npm run dev
